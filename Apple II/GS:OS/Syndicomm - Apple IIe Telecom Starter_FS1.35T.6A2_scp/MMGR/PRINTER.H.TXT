 PRINTER HELPYou can toggle your printer on or off by using the P command in the terminalcommand mode.  Your printer will print text received while you are online, textviewed from the capture buffer, or text read from a disk file to the screen. If you are printing while viewing the capture buffer, you may also toggle theprinter off or on by typing a [CTRL] P.If you have specified an imbedded printer control character in the INSTALLprogram, you can toggle the printer on or off by placing the control characterin the text.  You must first toggle the printer on by using the P command inthe terminal command mode.  Whenever an imbedded printer control characterappears in the text, it will toggle the printer off or on.  If you don't wantthe imbedded printer control character feature, specify $FF as the controlcharacter in the INSTALL program.
