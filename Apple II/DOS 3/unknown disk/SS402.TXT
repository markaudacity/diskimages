*TEXT SCREEN DUMP COMMANDS CAN BE
*EASILY TYPED FROM THE KEYBOARD OR SENT
*UNDER PROGRAM CONTROL.
* 
*FROM THE KEYBOARD- <CTRL-I> 2S
* 
*FROM A PROGRAM- PRINT CHR$(9); "2S"
* 
*THE GRAFSTAR // SUPPORTS FOUR
*DIFFERENT SCREEN DUMP COMMANDS AND
*THESE ARE DESCRIBED BELOW.
* 
*<CTRL-I> S  APPLE 40 COLUMNS
*<CTRL-I> 2S 40 COLUMN SIDE BY SIDE
*<CTRL-I> 3S VIDEX 80 COLUMN DISPLAY
*<CTRL-I> 8S APPLE //E 80 COLUMNS
* 
* 
*  <1>     VIEW TEXT PAGE 1
*  <SPACE> PRINT SCREENS SIDE BY SIDE
*  <ESC>   RETURN TO MAIN MENU
